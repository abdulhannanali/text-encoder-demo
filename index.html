Text Encoder and Decoder Demo

This is a demo built to present the function of TextEncoder and TextDecoder made to easily Encode the stream of Code points into stream of bytes and decoded the stream of bytes into stream of code points. This is a simple to use Encoding and Decoding application which allows to convert from the supported encodings.

TextEncoder Demo

TextEncoder takes a stream of code points and emits a stream of bytes. You can encode any string value and the output of this will be an encoded stream of Uint8Array you can use within applications. The output can be pasted into the decoder to get the decoded string back, given the encoding matches.

Decoder Demo

TextDecoder decodes the stream of bytes in a Uint8Array to stream of code points, output is returned as a string. The default encoding is the most common one used in JS world known as utf-8.
The input here accepts a comma separated string, of byte values that needs to be converted. An array formatted value with square brackets at the sides is considered valid value to use, as the values are extracted.
